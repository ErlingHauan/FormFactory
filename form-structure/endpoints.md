## API Endpoints

### Sign up
POST
/api/auth/signup

### Login
POST
/api/auth/login

### Get all forms
GET
/api/forms

### Create new form
POST
/api/forms

### Get specific form
GET
/api/forms/:id

### Delete specific form
DELETE
/api/forms/:id

### Get all responses for specific form
GET
/api/forms/:id/responses

### Create new response
POST
/api/forms/:id/responses
